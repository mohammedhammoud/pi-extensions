# @mh/pi-worker

Pi extension for spawning isolated worker processes.

## Features

- `/worker` command
- Dynamic model selection (defaults to current model)
- Isolated `pi` process spawn
- Compact final report

## Install

From npm:

```bash
pi install npm:@mh/pi-worker
```

## Development

```bash
pnpm install
pnpm run typecheck
pnpm run build
```
